Concurrent GPU Hash Tables

Authors: Hunter McCoy, Prashant Pandey

Status: Under Review

Abstract

Implemented a new suite of concurrent, highly efficient GPU Hash tables. Showed that external synchronization is necessary for correctness on insertions and deletions, but that specialized load instruction can perform memory safe lock-free loads on the GPU. These tables have peak performance of over 4 billion operations per second while still allowing for complete sequential memory correctness.
