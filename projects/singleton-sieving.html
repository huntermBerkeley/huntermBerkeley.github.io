Abstract

Integrated our GPU filters into a massively parallel GPU application, the de-novo metagenomic assembler MetaHipMer2. This resulted in an up to 5.4 times reduction in memory use in the k-mer analysis phase and a 40% reduction in total memory use across the entire assembler.
