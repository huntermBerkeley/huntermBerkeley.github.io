Gallatin: A General-Purpose GPU Memory Manager

Add your representative image here

Authors: Hunter McCoy, Prashant Pandey

Venue: PPOPP 2024

Abstract

Designed a concurrent GPU memory Manager Gallatin, which uses a parallel ensemble of van Emde Boas (vEB) trees to efficiently partition GPU memory. Automatic thread coalescing and a three-tiered slabbing technique allow for small allocations to be given at up to 12.6 billion allocations per second. This results in a GPU allocator with raw throughput up to 374 times faster than the state of the art, translating to 1.5 times performance gains on real-world dynamic graph benchmarking.
